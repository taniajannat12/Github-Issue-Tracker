1️⃣ What is the difference between var, let, and const?
ANS: var is legacy, let allows reassignment, and const is for fixed values.
2️⃣ What is the spread operator (...)?
ANS: copy or marge in array or object
3️⃣ What is the difference between map(), filter(), and forEach()?
ANS: map transforms to a new array, filter creates a subset, and forEach just iterates.
4️⃣ What is an arrow function?
ANS: modern way to write functions using =>. such as: spinner=()=>{....}
5️⃣ What are template literals?
ANS: Using backticks (`) to easily embed variables inside strings with ${}.
